# Day 7 - Random Floating Circles

## Project Overview
Interactive web app that generates colorful circles with random properties and smooth animations.

## Features
- Random circle generation (size, position, color)
- Automatic removal after random intervals
- Smooth scale-down animations
- Responsive design
- Dark theme with hover effects

## Tech Stack
- HTML5
- SCSS/CSS3
- JavaScript (ES6)

## Key Concepts
- DOM manipulation
- Random number generation
- CSS transitions
- Event handling
- setTimeout functions

## How to Use
1. Click the button to generate circles
2. Circles appear randomly on screen
3. They automatically disappear with animations
4. Continue clicking for more circles
