# Day 3 - Functions Level 1

## Exercises (10 Questions)

### Level 1 - Function Fundamentals
1. **sayHello()** - Print "Hello JavaScript!" using function
2. **add(a, b)** - Return sum of two numbers
3. **greet(name)** - Function with default parameter "Guest"
4. **addUnlimited(...numbers)** - Add unlimited numbers using rest parameters
5. **IIFE** - Immediately Invoked Function Expression
6. **Nested Functions** - Inner function accessing outer variable
7. **Array Manipulation** - Add/remove elements from fruits array
8. **Loop Through Array** - Print all elements using forEach
9. **Object Iteration** - Print all values of person object
10. **setTimeout** - Execute function after 2 seconds delay

## Key Concepts Practiced
- Function declaration and invocation
- Parameters and arguments
- Default parameters and rest parameters
- IIFE (Immediately Invoked Function Expressions)
- Nested functions and scope
- Array methods (push, shift, forEach)
- Object iteration with for...in loop
- setTimeout for delayed execution
- Return values and console logging
