# Day 4 - Functions Level 2

## Exercises (10 Questions)

### Level 2 - Advanced Function Concepts
1. **Higher-Order Functions** - `runTwice(fn)` that executes a function two times
2. **Pure vs Impure Functions** - Demonstrate both with examples
3. **Object Destructuring in Parameters** - Extract `name` and `age` from object parameter
4. **`this` Context Difference** - Normal function vs arrow function as object methods
5. **Array `map()` Method** - Create new array with squared numbers
6. **Array `filter()` Method** - Get only even numbers from array
7. **Array `reduce()` Method** - Calculate total salary from array of numbers
8. **Array `some()` and `every()` Methods** - Test conditions on array of names
9. **Object Immutability** - Test `Object.freeze()` and `Object.seal()` behavior
10. **Nested Object Access** - Access city from nested object using destructuring
